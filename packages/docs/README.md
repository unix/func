# Func Docs

Astro documentation project for `func`.

## Commands

```sh
pnpm dev
pnpm build
pnpm preview
pnpm deploy:dry-run
pnpm deploy
pnpm format
```

The project uses Astro, TypeScript, Tailwind CSS v4, shadcn with baseUI, and the
`@norehq/prettier` shared rules. Deployment uses Cloudflare Workers Static
Assets via Wrangler.
